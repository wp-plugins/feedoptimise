WooCommerce extension enabling direct integration with feedoptimise.com services.

== Description ==

This plug-in lets your WooCommerce store easily integrate with <a target="_blank" href="https://www.feedoptimise.com/">FeedOptimise.com</a> by enabling data feed exporting capabilities and inserting click and conversion tracking tags.

== Installation ==

1. Upload the Feed Optimise plugin directory into your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Get your Feed Optimise's Customer ID.
4. In the WordPress admin panel go to settings and then click on Feed Optimise. Please enter your Customer ID under Customer ID field. 
5. Delete the cache on your server and on your browser.
